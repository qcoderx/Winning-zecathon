# PulseFi Frontend

The complete landing page for PulseFi - The Trust Engine for SME Lending.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `/src/components/` - React components (Header, Hero, Features, Process, Opportunities, Footer)
- `/src/App.jsx` - Main application component
- `/src/App.css` - Custom styles and animations
- `/src/index.css` - Tailwind CSS imports

## Key Features

- Complete landing page with all sections
- Animated heartbeat SVG in hero section
- Responsive design with Tailwind CSS
- Custom gradient text and backgrounds
- Material Symbols icons
- Dark mode support
